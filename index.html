Step-by-Step Guide دليل خطوة بخطوة

1

Set Up Your API Key إعداد مفتاح API

Click the ⚙️ Settings icon in the top-right corner. Get a free API key from Google AI Studio, paste it in the settings, choose your preferred AI model, and click Save.

اضغط على أيقونة ⚙️ الإعدادات في الزاوية العلوية. احصل على مفتاح API مجاني من Google AI Studio، الصقه في الإعدادات، اختر نموذج الذكاء الاصطناعي المفضل، واضغط حفظ.

Your key is saved locally in your browser — you only need to do this once. يتم حفظ مفتاحك محلياً في متصفحك — تحتاج لفعل هذا مرة واحدة فقط.
2

Describe Your Form صِف النموذج المطلوب

In the input box below, write what you need in plain language. Be specific about question types:

في مربع الإدخال أدناه، اكتب ما تحتاجه بلغة عادية. كن محدداً بشأن أنواع الأسئلة:

Create a feedback form with: name (required), email (required), satisfaction 1-5 (multiple choice), services used (checkboxes: Support, Sales, Billing), and comments (optional paragraph) أنشئ نموذج ملاحظات يحتوي على: الاسم (مطلوب)، البريد الإلكتروني (مطلوب)، تقييم الرضا 1-5 (اختيار متعدد)، الخدمات المستخدمة (خانات اختيار: الدعم، المبيعات، الفواتير)، والتعليقات (فقرة اختيارية)
3

Generate Table & Code إنشاء الجدول والسكريبت

Click ✨ Generate Form (or press Ctrl+Enter). The AI will produce two outputs:

اضغط ✨ إنشاء النموذج (أو اضغط Ctrl+Enter). سينتج الذكاء الاصطناعي مخرجين:

Table — Your form structure for Google Sheets جدول — هيكل النموذج لجداول Google
Script — Apps Script code to create the form سكريبت — كود Apps Script لإنشاء النموذج
4

Paste the Table into Google Sheets الصق الجدول في جداول Google

  1. Open Google Sheets → create a new blank spreadsheet
  2. Click 📋 Copy Table in the app
  3. Click cell A1 in your spreadsheet
  4. Paste with Ctrl+V
  1. افتح جداول Google ← أنشئ جدول بيانات فارغ جديد
  2. اضغط 📋 نسخ الجدول في التطبيق
  3. اضغط على الخلية A1 في جدول البيانات
  4. الصق باستخدام Ctrl+V
5

Add the Apps Script إضافة سكريبت Apps Script

  1. In your Google Sheet, go to Extensions → Apps Script
  2. Delete all default code in the editor
  3. Click 📋 Copy Code in the app
  4. Paste the code into the Apps Script editor
  5. Click 💾 Save (or Ctrl+S)
  1. في جدول Google، اذهب إلى الإضافات ← Apps Script
  2. احذف كل الكود الافتراضي في المحرر
  3. اضغط 📋 نسخ السكريبت في التطبيق
  4. الصق الكود في محرر Apps Script
  5. اضغط 💾 حفظ (أو Ctrl+S)
6

Run & Get Your Form! شغّل واحصل على النموذج!

  1. Select createFormFromSheet from the function dropdown
  2. Click ▶️ Run
  3. Authorize the script when prompted (first time only)
  4. Go to View → Logs to find your form's URL
  1. اختر createFormFromSheet من قائمة الدوال
  2. اضغط ▶️ تشغيل
  3. قم بتفويض السكريبت عند المطالبة (أول مرة فقط)
  4. اذهب إلى عرض ← السجلات للعثور على رابط النموذج
Your Google Form is now live! Click "Send" in the Forms editor to share it. نموذج Google جاهز الآن! اضغط "إرسال" في محرر النماذج لمشاركته.
